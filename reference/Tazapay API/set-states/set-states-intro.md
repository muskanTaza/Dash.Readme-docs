This group of API calls can help you to set different object states for transactions (checkout and escrow) and KYB applications.

For more info on object states, refer to these pages:

- [checkout object states](https://docs.tazapay.com/docs/checkout-object-states) 
- [escrow object states](https://docs.tazapay.com/docs/escrow-object-states)
- [KYB object states](https://docs.tazapay.com/docs/kyb-object-states)

> 📘 These API URLs are different from Sandbox's URL
> 
> Currently, the following functions use a different URL as such:  
> `https://gxyhxwtbx8.execute-api.ap-southeast-1.amazonaws.com/sandbox/external` 
> 
> This is different from `https://api-sandbox.tazapay.com` as these functions are solely for use in sandbox and not in production.
