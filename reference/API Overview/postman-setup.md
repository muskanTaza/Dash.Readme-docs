Postman is an API platform for building and using APIs. Postman simplifies each step of the API lifecycle and streamlines collaboration so you can create better APIs—faster.

### **How to install Postman**

Postman is available on the web at go.postman.co/home and as a native desktop app for Mac (Intel and M1), Windows (32-bit / 64-bit), and Linux (64-bit) operating systems.

Get the latest version of <a href="https://www.postman.com/downloads/" target="_blank">Postman</a> on their downloads page.

### **Importing our Postman Collection**

Once you have installed Postman, you can then <a href="https://github.com/tazapay/postman-collections" target="_blank">download Tazapay's Postman Collection</a>
