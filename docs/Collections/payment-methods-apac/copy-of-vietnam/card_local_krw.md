# How It Works

* Users pay using locally issued South Korean debit or credit cards denominated in KRW.
* Transactions are processed through domestic card networks, ensuring high success rates and compliance with local regulations.
* Payments are completed via standard card flows, including card details entry and issuer authentication where applicable.

# Payment Method type

Users can use the below payment method type to create a transaction using Korean Cards in the available geography

* South Korea: `card_local_krw`

# Limits and Expiration

The following is the upper and lower limits for Korean Cards transactions:

* **Minimum Limit**: USD 0.1
* **Maximum Limit**: USD 15000
