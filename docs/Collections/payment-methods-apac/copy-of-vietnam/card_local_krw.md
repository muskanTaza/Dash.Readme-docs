# How It Works

* Users link their Vietnamese bank accounts or top up their ShopeePay wallets.
* Primarily used for seamless and secure transactions within the Shopee app but also accepted by a growing number of offline merchants.
* Payments are made through the app, often via QR code scanning.

# Payment Method type

Users can use the below payment method type to create a transaction using ShopeePay in the available geography

* Vietnam: `shopeepay_vnd`

# Functionality

| Functionality               | Yes/No | Description                                                                                                    |

| **Generally Available**     | ✅      | Means that the payment method is available by default and does not need any additional configuration/approval. |
| **Dispute and Chargebacks** | ❌      |                                                                                                                |

# Limits and Expiration

The following is the upper and lower limits for ShopeePay transactions:

* **Minimum Limit**: USD 0.07
* **Maximum Limit**: USD 1800

The default expiration for a transaction is **25 minutes.**
