# How It Works

* Users link their Korean bank accounts or cards to their Payco wallets.
* Payments are authorized within the Payco app using PIN or biometric authentication.
* Commonly used for seamless transactions across e-commerce platforms and partner merchants.

<br />

# Payment Method type

Users can use the below payment method type to create a transaction using KakaoPay in the available geography

* South Korea: `payco_krw`
