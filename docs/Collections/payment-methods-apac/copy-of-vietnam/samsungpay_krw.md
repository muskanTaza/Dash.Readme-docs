# How It Works

* Users add their Korean-issued debit or credit cards to Samsung Pay on supported Samsung devices.
* Payments are authenticated using biometric verification or device PIN.
* Transactions are completed via in-app payment flows or supported merchant checkout experiences.

# Payment Method type

Users can use the below payment method type to create a transaction using Samsung Pay in the available geography

* South Korea: `samsungpay_krw`
