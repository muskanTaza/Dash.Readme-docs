# How It Works

* Users link their Korean bank accounts or cards to their KakaoPay wallets.
* Payments are authorized within the KakaoPay app, often using biometric or PIN-based authentication.
* Commonly used for fast, secure transactions across e-commerce, apps, and partner merchants

  <br />

  # Payment Method type

  Users can use the below payment method type to create a transaction using KakaoPay in the available geography

  * South Korea: `kakaopay_krw`
