<h2>Settlement Overview</h2>
<p><strong>Settlement</strong> is the process by which Tazapay transfers the funds from completed transactions to the merchants. By default, this happens on a weekly basis, directly into the bank accounts that merchants have provided in their dashboard settings or during the KYB process, which can be managed and viewed <a href="https://dashboard.tazapay.com/settings/bank">here</a>.</p>

<h3>How Do You Know When a Settlement Has Been Created?</h3>
<ol>
  <li><strong>Email Notification</strong>: 
    <ul>
      <li>As soon as a settlement file is created, Tazapay sends an email to the merchant's account with important details:</li>
      <ul>
        <li><strong>Settlement ID</strong>: Unique identifier for the settlement.</li>
        <li><strong>Settlement Amount</strong>: Total amount to be settled.</li>
        <li><strong>Date of Creation</strong>: When the settlement file was created.</li>
        <li><strong>Link to View Settlement</strong>: Direct link to view the details of the settlement. This can often help merchants quickly verify and manage settlements directly.</li>
      </ul>
    </ul>
  </li>
  <li><strong>Dashboard Report</strong>:
    <ul>
      <li>A detailed settlement report is also generated and can be accessed on the dashboard <a href="https://dashboard.tazapay.com/settlement/">here</a>. Merchants can download this report to view comprehensive details of the settlements.</li>
      <li>To understand all the fields in the settlement report, refer to this <a href="https://support.tazapay.com/what-do-the-columns-in-the-exported-csv-file-mean">FAQ</a>.</li>
      <li>Download a sample settlement report <a href="https://7971209.fs1.hubspotusercontent-na1.net/hubfs/7971209/Settlement%20Report%20Sample.xlsx">here</a>.</li>
    </ul>
  </li>
</ol>

<h3>Importance of the Settlement Report for Reconciliation</h3>
<p>The settlement report is crucial for merchants' financial reconciliation processes. It provides detailed insights into each transaction and settlement, helping merchants verify and reconcile their records accurately. For a deeper understanding of how this ties into overall financial reconciliation, visit the <a href="https://docs.tazapay.com/docs/reconciliation">Reconciliation Documentation</a>.</p>
