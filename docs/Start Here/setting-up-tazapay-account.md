To get started with Tazapay, you will need to create an account on the <Anchor label="sandbox (test) environment" target="_blank" href="https://dashboard-sandbox.tazapay.com/signup">sandbox (test) environment</Anchor>.

# Access your API keys

Once the sandbox account is active, fetch your API keys to call Tazapay's endpoints.

1. Log in to the <Anchor label="Tazapay Sandbox Dashboard" target="_blank" href="https://dashboard-sandbox.tazapay.com">Tazapay Sandbox Dashboard</Anchor>.
2. On the left hand panel, select Settings.
3. Navigate to Integration -> API Keys and Developer Docs.
4. You should be able to view the keys.
