A refund occurs when a buyer requests to have their transaction cancelled and to have their payment refunded to their account.

If a refund is requested, Tazapay will refund the amount to the source used by the buyer to make the payment.
[block:callout]
{
  "type": "warning",
  "body": "The buyer must have made payment into the Tazapay [payment link](doc:payment-links-core-concept) before a refund can be initiated."
}
[/block]
Try the [Initiate Refund](ref:refund-api) API here.
